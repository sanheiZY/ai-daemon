# AI后台服务

本项目基于LangChain框架开发，旨在提供一个高效、可扩展的AI后台服务，支持通过DBus接口与外部系统进行交互。服务内集成了简单的Agent和RAG机制，能够满足不同场景下的需求。

## 特性

- **简单Agent**：集成基础智能代理，支持自定义任务处理。
- **RAG支持**：通过检索增强生成技术提升回答的质量与准确性。
- **DBus接口**：提供DBus接口，便于与现有系统无缝集成。
